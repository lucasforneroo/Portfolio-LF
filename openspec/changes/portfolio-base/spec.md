## Spec: portfolio-base

### Functional Requirements
- Responsive Radial Layout.
- SPA Navigation.
- High-impact visual nodes.

### Non-Functional Requirements
- LCP < 2.5s.
- Type Safety (TS).
- Utility-First Styling (Tailwind v4).

### Implementation Scenarios
- Scenario 1: Initial Scaffolding and Tailwind v4 config.
- Scenario 2: Folder structure creation.
- Scenario 3: Base Theme (Neon/Cyberpunk) implementation.
- Scenario 4: Radial Navigation Shell.
